# 用高雄的氣候資訊來預測高雄的用電量
使用線性迴歸

## Requirement
- ruby 2.5.0
  - nokogiri
  - pycall
- python 3.6.5
  - numpy
  - scipy
  - sk-learn
  - matplotlib

## 資料來源
- source 1 每月氣象-過去9年局屬地面測站每月氣象資料: https://data.gov.tw/dataset/23827
- source 2 台灣電力公司_鄉鎮市(郵遞區)別用電統計資料: https://data.gov.tw/dataset/14135
